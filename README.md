# CTRL-SHORTCUTS

## Functions

CTRL + A = Select all
CTRL + B = Bold
CTRL + C = Copy
CTRL + ALT + V = Paste Special
CTRL + D = Fill Down
CTRL + F = Find
CTRL + G = Go To
CTRL + H = Replace
CTRL + I = Italic
CTRL + K = Insert Hyperlink
CTRL + N = New Workbook
CTRL + O = Open File
CTRL + P = Print
CTRL + R = Fill Right
CTRL + S = Save Workbook
CTRL + T = Create Table
CTRL + U = Underline
CTRL + V = Paste
CTRL + W = Close Window
CTRL + X = Cut
CTRL + Y = Repeat
CTRL + Z = Undo
CTRL + 1 = Format Box
CTRL + 5 = Strike - through
CTRL + 9 = Hide Row
SHIFT + CTRL + 9 = Hide row
CTRL + 0 = Hide column
SHIFT + CTRL + 0 = Unhide column
CTRL + ; Display Date
SHIFT + CTRL + : Display Time
CTRL + Space = Select column
CTRL + Enter = Fill selection with entry
